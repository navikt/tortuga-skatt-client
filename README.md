Tortuga Skatt Client
====================

Inneholder klientbiblioteker for å kommunisere med Skatteetaten sine API-er.

## Installasjon og kjøring

### Bygging

For å bygge JAR:

```
make
```

Man kan også bygge med maven:

```
mvn package
```
